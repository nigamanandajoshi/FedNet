# DEPLOY FEDNET TO REPLIT (3 MINUTES)

## Step-by-Step Deployment

### Step 1: Go to Replit
Visit: **https://replit.com**

### Step 2: Create New Repl
1. Click **"+ Create"** → **"Import from GitHub"**
2. Paste your repo URL:
   ```
   https://github.com/nigamanandajoshi/FedNet
   ```
3. Click **"Import"**

### Step 3: Wait for Import (1 minute)
Replit clones your repo automatically.

### Step 4: Create run.sh
In the Replit file editor, create a new file called `run.sh`:

```bash
#!/bin/bash
python app_simple.py
```

### Step 5: Click Run
Click the **"Run"** button (top center).

Replit will:
- Auto-detect Python
- Install dependencies from requirements.txt
- Start your Flask app
- Generate a live URL

### Step 6: Get Your Live URL
You'll see output like:
```
* Running on http://0.0.0.0:5000
```

Replit will show a **live URL** at the top:
```
https://fednet-xxxxx.replit.dev
```

**This is your live dashboard!** 🎉
